== Description ==

Allows you to show your NaNoWriMo Stats in posts, pages and sidebar widgets. Also, NaNoWriMo widgets are available.

The NaNo Stats plugin comes with clear explanation and documentation.

This is an unofficial plugin, not made by the Office of Letters and Light.

== Installation ==

1. Upload folder `nano-stats` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is NaNoWriMo? =
NaNoWriMo is short for National Novel Writing Month. It is a worldwide contest for (hobby)writers to write 50,000 words in one month, November. For more information, please visit http://nanowrimo.org
